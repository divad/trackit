Version Control

Need to store your software or project somewhere? Need to store/track the changes you make? Soton Forge can help! You can choose from Apache Subversion, Git or Mercurial to store your files in.

Self Provisioning

Create and manage teams of people, and within the teams, create as many repositories as you need. Each repository can have individual permissions and different revision control and web management tools

Project Management

Use either Trac or Redmine to manage your project. Both tools optionally integrate with Subversion, Git and Mercurial. The tools offer features such as bug/ticket tracking, a wiki, milestones and other features
