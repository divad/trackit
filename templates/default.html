Version Control

Need to store your software or project somewhere? Need to store/track the changes you make? Soton Forge can help! You can choose from Git or Subversion to store your files in. We'll be adding Mercurial support soon.

Self Provisioning

Create and manage teams of people, and within the teams, create as many repositories as you need. Each repository can have individual permissions and different revision control and web management tools

Project Management

Use can use Trac to manage your project. Trac optionally integrates with Subversion and Git. Trac offers features such as bug/ticket tracking, a wiki, milestones and other project management features

Get full control over your project

The Forge system puts you in full control. You decide what type of revision control system to use, what project management tool to use and who gets access to your project. You manage all the settings and permissions on all your projects, and changes are reflected immediatley without having to wait for iSolutions.

Manage your project with Trac!

Trac makes it easy to manage projects - software projects or otherwise. iSolutions itself uses Trac to manage its internal projects. It features task management (which can be used for bug tracking), a full wiki for documentation, a 'roadmap' system for managing sub-groups of tasks and other features such as task completion tracking, gantt tracking, etc.

Its not all command line!

Although both SVN and Git come from the command line world there are no shortage of easy to use and amazing GUI tools for both. For Git we recommend TortoiseGit (Windows), SourceTree (Windows, Mac OS X), GitX-dev (Mac OS X) and GitG (Linux). For Subversion we recommend either TortoiseSVN (Windows) or RapidSVN (Windows, Mac, Linux).
